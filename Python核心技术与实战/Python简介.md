# 前言

Python可以运用在数据处理、Web开发、人工智能等多个领域。语言简洁、开发效率高、可移植性强，并可和其他语言轻松无缝衔接。

阅读Python源码，主要是CPython。

从工程角度学习Python。



很多公司服务器端开发基于Python，但底层的基础架构依赖于C++。

对于技术选择要有独立的思考：为什么选择这个技术？这个技术解决了哪些问题？为什么不选择别的技术？有哪些局限？
